# Presentation Slides

This directory is dedicated to storing and sharing the group's presentation slides.

## How to Add Your Slides

1. Add your PowerPoint (`.pptx`) or PDF presentation file to this directory.
2. Use a descriptive filename, for example:  
   `YYYY-MM-DD_<topic-or-conference-name>.pptx`
3. Commit and push your changes (or open a Pull Request).
